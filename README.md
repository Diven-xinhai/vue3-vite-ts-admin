# 项目介绍
项目名称：Vue3-Vite-Ts-admin

核心思想与若依框架相似，熟悉的同学可以快速上手~

本项目目前处于完成骨架状态，可以更好的根据自已业务需求进行开发~

TypeScript部分运用的语法目前并未使用高阶语法，后期可能会不断更新...
## 技术栈：

- Vue3
- Vite
- TypeScript
- Echarts
- ElementUI
- Sass
